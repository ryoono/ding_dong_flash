# ding_dong_flash

インターホンの音が聞こえない人向けに、来客を光で報告。  
本親デバイスを、インターホン親機に接続し、来客情報を離れた場所にある子デバイスに送信する。  
来客情報を受信した子デバイスは、ハイパワーLEDを点滅させ、来客を知らせる。  
商品名は「ピンポンフラッシュ」、ピンポンダッシュをもじった。
